P&ensp;&ensp;&ensp;&ensp;&ensp;&ensp; - Pause in-game <br />
Space&ensp;&ensp;   - Shop in-game <br />
M &ensp;&ensp;&ensp;&ensp;&ensp;      - Stop music but not button sfx <br />
ESC&ensp;&ensp;&ensp;&ensp;     - Exit game in any state <br />
